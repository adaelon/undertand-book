# ADR-0006 E 的记忆层:用户私有、跨书、两层,与只读基座物理隔离

状态:已接受(2026-06-21,grill 共识)

## 背景
[[ADR-0005]] 定 E 有 memory。约束来自 V2 §3.4(基座导入一次即冻结、只读)与 §3(原文/补充物理隔离)。

## 决策
1. **物理隔离(定理)**:memory **绝不写进冻结的 `.understand-book` 只读基座**,落在一个**独立、用户私有、可变的记忆层**。只读基座因此可分发 / 多人共享而不被个人数据污染。
2. **作用域 = 用户全局、跨书**:记忆挂在"用户"下、跨所有读过的书。book agent 读写它但**不拥有**它。⇒ 把 V2 §5.4 跨书协议从"预留"变成**记忆层的自然能力**("你在《X》问过的概念,《Y》又出现了")。
3. **两层**:
   - **会话工作记忆**(临时):当前对话 + 阅读位置。
   - **长期记忆**(持久):阅读旅程、历次问答、兴趣、卡点、用户笔记/高亮。

## 命门
跨书时需做**实体/概念对齐**——不同书对同一概念的 LID 不同;靠 LID 永久有效(§3.4)+ 一层概念映射。

## 否决
- **每书一份记忆**:丢掉跨书增量,也浪费 §5.4。
- **记忆写进只读基座**:违 §3 隔离,毁掉"基座可分发"。

## 影响
新增"用户记忆层"的存储与读写契约。V2 §5.4 跨书协议落到记忆层实现。
