# ADR-0020 记忆迁移(议题8 第二叉/8b):消费 8a 确定性 LID diff + 三命运重锚 + v1 记忆不改写 + 读时确定性投影非物化 + 禁猜最近邻

状态:已接受(2026-06-22,工程层 grill 议题8 第二叉共识)

## 背景
议题8 第二叉(8b)。[[ADR-0019]](8a)定增量构建产出**独立新 book_id 基座(v2)、旧基座(v1)只读并存**。但用户在 v1 上积累了记忆——`note/highlight/qa`(显式 save,带 `citations:[{lid,book_id}]`,[[ADR-0015]])、`interest/sticking_point/journey`(Phase1 推断,[[ADR-0018]])、`position`(阅读位置)。改版后打开 v2,这些记忆怎么跟过来?难点:v1 的 `citation.lid` 在 v2 有三种命运——稳定 / 原文漂移 / 失效;而 [[ADR-0015]] 禁宽松降级(绝不自动重锚到最近邻 LID,毒化引用红线)。最高原则(正确性优先、只喂确定性事实+真原文、禁宽松降级)贯穿。本叉收口议题8。

## 决策
1. **重锚真相源 = 消费 8a 确定性 LID diff,概念对齐退居兜底**:v1→v2 是**同一本书增量**,8a 的 Merkle diff + 变更分级([[ADR-0019]])**确定性地知道** LID 演化 ⇒ 产出
   ```
   lid_migration_map: v1_lid → { status: stable | content_drift | removed, v2_lid? }
   ```
   重锚锚定此确定性映射。**否决退回 [[ADR-0006]] 概念层模糊对齐**——那是为「真不同的两本书」(无 LID 对应关系)准备的;同书增量有确定性 diff,用模糊对齐是降精度、违正确性。概念对齐仅当某记忆锚 `{concept}` 而非 `{lid}` 时兜底。
2. **引用红线在迁移中从不破**(地基):记忆 citation 带 `book_id`([[ADR-0015]])+ v1 只读并存([[ADR-0019]]) ⇒ 一条 citation 永远指向真实存在的 `(book_id, lid)`,**失效就诚实锚回 v1 历史版本,不猜、不降级**。这是全叉能守 [[ADR-0004]]/[[ADR-0015]] 的物理基础。
3. **三命运确定性重锚**(无 LLM,守 B2):
   | 命运 | 动作 |
   | --- | --- |
   | **stable**(LID 在 v2 存在且原文等价) | 投影 `{lid:v2_lid, book_id:v2}` |
   | **drift**(LID 路径同、原文被改字) | 投影 `{lid:v2_lid, book_id:v2, drift:true}`(标「源文已变·需复核」,认知诚实 [[ADR-0018]])+ **保留 v1 锚**可回溯原始原文 |
   | **removed**(STRUCTURAL,v2 无对应 LID) | **不投影 v2、不猜最近邻**;标 `v2_status:orphaned`,citation 仍指 v1(可跳 v1 原文)|
4. **v1 记忆库永不改写**:用户当初在 v1 的锚是**历史事实**,不因改版被改写(认知诚实);v1 只读并存 ⇒ v1 记忆原样保留、永远可验证、是唯一真相源。
5. **迁移 = recall 时确定性投影,不物化新记忆库**(呼应 [[ADR-0012]]/[[ADR-0013]] 不物化派生视图):
   ```
   recall(book_id:v2, lid):
     hits = 召回 citation.lid 经 map 命中 lid 的 v1 记忆
     for h in hits: project(h, lid_migration_map)   # 确定性查表、毫秒级,按三命运表
   ```
   **不在「打开 v2」时批量改写出 v2 记忆库**——物化会重蹈「胶囊↔图谱一致性」负担([[ADR-0012]]);读时投影零一致性风险、v1 记忆唯一真相源。
6. **removed 记忆 recall@v2 默认返回 + 标 orphaned**(不隐藏):最高原则「agent 上下文完全正确」⇒ 隐藏失锚记忆 = 给 agent 不完整视图;让 agent 看到「用户旧版此处有卡点、新版该段被删/改」是有价值信号(卡点或正被改版解决)。代价(recall 混入少量历史锚)由 `orphaned` 标显式隔离。
7. **来源三分迁移分裂 + 会话层不迁移**(确定性应用同一 map,[[ADR-0018]]):
   - **显式 save**(`note/highlight/qa`,有 `citations[lid]`)→ 走 map 投影(三命运)。
   - **Phase1 推断**(`interest/sticking_point/journey`)→ `{concept}` 锚走 [[ADR-0006]] 概念兜底;其 lid 证据走 map。
   - **会话临时层**(`position/dialogue`,session)→ **不迁移**(改版重读 = 新会话,从新位置起)。

## 命门
- **citation 带 book_id + v1 只读并存 = 引用红线不破的物理前提**:没有 v1 并存,removed 就无处锚、被迫降级;有了它,失效=锚回历史,而非丢失。
- **消费 8a diff 而非概念对齐**:同书增量的确定性 diff 是 8a 白送的副产物,8b 不重新猜 LID 对应,精度=确定性。
- **读时投影非物化** = 项目一贯哲学第三次落地([[ADR-0012]] 语境胶囊 / [[ADR-0013]] book.context / 本叉记忆迁移):派生视图一律读时确定性投影,唯一真相源不复制。
- **三处认知诚实一致**(drift 标源文变 / removed 标 orphaned / Phase1 推断带证据标注 [[ADR-0018]]):根都是用户最高原则——agent 上下文必须完全正确,不掩盖不确定。

## 否决
- **退回 [[ADR-0006]] 概念对齐做主迁移路径**:同书增量有确定性 diff,模糊对齐是降精度。
- **removed 自动重锚最近邻 LID**:[[ADR-0015]] 明令禁宽松降级,静默给错锚毒化引用红线。
- **改写 v1 记忆库的 book_id/lid(替换式迁移)**:抹掉历史锚事实,违认知诚实;且 v1 记忆是 removed/drift 回溯的唯一来源,改写即失溯。
- **物化 v2 记忆库(批量迁移)**:胶囊↔图谱式一致性负担 + 双真相源漂移风险,撞 [[ADR-0012]]。
- **removed 记忆 recall@v2 隐藏**:给 agent 不完整视图,违最高原则;丢失「改版解决卡点」信号。
- **会话临时层也迁移 position**:改版重读是新会话,迁移旧位置无意义。
- **drift 静默重锚不标注**:用户笔记针对 v1 原文,v2 同位原文已改,不标=喂 agent 失配上下文。

## 何时回头
- 切片0 实测填:多次改版(v1→v2→v3)citation 是否膨胀——拟 recall 只取「当前版本有效锚 + 最早溯源」,中间版本压缩;实测改版频次后定压缩策略。
- 若 `drift` 判定过粗(改一字即标全 LID drift)导致 recall@v2 噪声高 → 评估 drift 内再分「实质改/排版改」,仍确定性(diff 字节级),不引 LLM。
- 跨书概念对齐([[ADR-0006]] 命门)实测若误聚 → 扩概念映射/实体消歧,不退回单维。
- `orphaned` 记忆是否需主动提示用户手动重锚/归档 → 读时交互分阶段定([[ADR-0005]] §5.4)。
